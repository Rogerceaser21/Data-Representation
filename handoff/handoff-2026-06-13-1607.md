## Build/tooling gotchas learned
- **clasp:** use `npx --yes @google/clasp` (bare clasp not installed).
- **Preview server** sandbox CANNOT read Google Drive paths — stage to `/tmp/<dir>` and serve via `sh -c "cd /tmp/... && python3 -m http.server PORT"`. It also THROTTLES rAF animation, so screenshots catch GSAP/three.js mid-animation; force final state via `gsap.globalTimeline.progress(1)` + manual style sets before screenshotting, or just trust the data and view locally.
- **gws sheets:** `batchUpdate` (camelCase) not `batch-update`; values write via `gws sheets spreadsheets values batchUpdate --params '{"spreadsheetId":...}' --json @payload` with `valueInputOption:USER_ENTERED`. Strip the `Using keyring backend` prefix before JSON-parsing.
- **Pages:** force build `gh api -X POST repos/Rogerceaser21/Data-Representation/pages/builds`; ~30-40s.
